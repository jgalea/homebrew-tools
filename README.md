# jgalea/homebrew-tools

Homebrew tap for command-line tools maintained by Jean Galea.

## Use

```sh
brew tap jgalea/tools
brew install cam
```

After tapping, install any formula from this tap with `brew install <name>`.

## Formulae

| Tool | Description |
|------|-------------|
| [cam](https://github.com/jgalea/cam-cli) | Viewfinder + capture from a Mac camera (incl. iPhone Continuity Camera) in the terminal. |
